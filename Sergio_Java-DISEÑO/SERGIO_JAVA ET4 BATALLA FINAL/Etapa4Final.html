Funciones con JavaScript

En terminos generales, una funcion es un "subprograma" que puede ser llmado por codigo externo (o interno en caso de recurion) a la funcion. Al igual que el programa en si mismo, una funcion se compone de una secuencia de declaraciones, que conforman el llamado cuerpo de la funcion. Se pueden pasar valores a una funcion, y la funcion puede devolver el valor
